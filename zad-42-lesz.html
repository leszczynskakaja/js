Pętle w JavaScript

Pętla for

Konstrukcja:

for (let i = 0; i < 10; i++) { }

Opis: Pętla z licznikiem.

Schemat blokowy: [for warunek blok inkrementacja]

Pętla while

Konstrukcja:

while (warunek) { }

Opis: Warunek przed wejściem.

Schemat blokowy: [warunek blok warunek]

Pętla do...while

Konstrukcja:

do { } while (warunek);

Opis: Warunek po wykonaniu.

Schemat blokowy: [blok → warunek → blok]
